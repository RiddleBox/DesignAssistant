## 已知限制

### 当前使用模拟模式
1. **模拟向量**：
   - 使用随机向量代替真实 embedding
   - 基于文档 ID hash 保证一致性
   - 可用于流程验证，但检索质量不代表真实效果

2. **模拟生成**：
   - 使用简单模板生成回答
   - 不调用真实 LLM API
   - 可用于流程验证，但回答质量不代表真实效果
