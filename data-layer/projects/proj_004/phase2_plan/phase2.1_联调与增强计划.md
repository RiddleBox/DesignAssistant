## 三、增强任务（联调稳定后）

### 增强A：Taxonomy 边界澄清（P1）

- **问题**：technical/market/team/capital 四类在边界案例上 Precision 不稳定
- **方案**：补充 2-3 个边界 few-shot 样例，明确排除规则
- **触发时机**：2.2 联调中发现因 taxonomy 模糊导致的评估错误

### 增强B：多信号聚合逻辑优化（P2）

- **问题**：同一事件多条信号时，去重/合并逻辑不稳定
- **方案**：在 decoder.py 后处理层加信号去重规则
- **触发时机**：端到端测试中出现重复信号干扰评估

### 增强C：Prompt → LLM 正式 API 接入（P1）

- **问题**：当前 decoder 为规则模拟，需接入真实 LLM
- **方案**：替换 decoder 核心调用为 Claude API
- **触发时机**：端到端真实案例运行前
