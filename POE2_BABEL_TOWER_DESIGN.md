# 🎮 POE2赛季设计方案：巴别塔·猎神之塔

> **设计日期**：2026-02-12  
> **设计团队**：3人核心团队（策划主导、系统设计师、数值设计师）  
> **设计阶段**：概念草案

---

## 📋 一、核心概念

### 1.1 一句话描述
**建造通天之塔，在倒计时结束前攀登到神域，猎杀神明，夺取神器。**

### 1.2 设计目标
1. **新鲜感**：赛季初提供全新的向上攀登体验
2. **留存驱动**：通过长期建造和挑战系统延长玩家游玩时间
3. **双重满足**：
   - 刷装备党：快速重复，大量掉落刺激
   - 做装备党：印记系统，可控性制造装备
4. **操作沉浸感**：倒计时压力 + 向上征服感 + 逃离机制

### 1.3 与现有系统的差异

| 维度 | 现有深渊 | 现有试炼 | 猎神之塔 |
|------|---------|---------|---------|
| 方向 | 向下 | 横向 | **向上** |
| 时间压力 | 无 | 无 | **倒计时** |
| 建造系统 | 无 | 无 | **自建塔** |
| 撤离方式 | 手动 | 手动 | **自动传送门** |
| 装备制作 | 无 | 无 | **印记系统** |

## 🏗️ 二、塔的建造系统

### 2.1 塔的核心属性（仅2个）

#### 🛡️ 塔身防御力
- **作用**：决定怪物从第几层开始能破防
- **示例**：
  - 防御力 30 → 30层以下怪物无法破防，31层开始破防
  - 防御力 60 → 60层以下安全，61层开始破防
  - 防御力 90 → 90层以下安全，91层开始破防

#### ⏰ 塔身耐久（倒计时）
- **作用**：破防后塔能支撑多久
- **示例**：
  - 耐久 20分钟 → 破防后20分钟塔倒塌
  - 耐久 40分钟 → 破防后40分钟塔倒塌
  - 耐久 70分钟 → 破防后70分钟塔倒塌

### 2.2 材料系统（POE风格）

#### 材料来源
```
普通怪物 → 掉落"神罚碎片"（大量）
精英怪物 → 掉落"神罚精华"（少量）
BOSS怪物 → 掉落"神罚核心"（稀有）
```

#### 兑换建造（NPC处）
```
碎片×1000 → 建造"基础塔"（防御30，耐久20分钟）
精华×100  → 建造"强化塔"（防御60，耐久40分钟）
核心×10   → 建造"神话塔"（防御90，耐久70分钟）
```

#### 升级塔
```
基础塔 + 精华×50 → 强化塔
强化塔 + 核心×5  → 神话塔
```

### 2.3 塔的等级表（数值示意）

| 塔等级 | 材料需求 | 防御力 | 耐久时间 | 适合目标层数 |
|--------|---------|--------|---------|-------------|
| 基础塔 | 碎片×1000 | 30层 | 20分钟 | 40-50层 |
| 强化塔 | 精华×100 | 60层 | 40分钟 | 70-80层 |
| 神话塔 | 核心×10 | 90层 | 70分钟 | 100层（神明） |

## ⚔️ 三、攀登与战斗机制

### 3.1 攀登流程

```
1. 建造塔（消耗材料）
   ↓
2. 进入塔，开始攀登
   ↓
3. 每层战斗，获得奖励
   ↓
4. 到达塔的防御力层数后，倒计时开始
   ↓
5. 在倒计时结束前尽可能爬高
   ↓
6. 时间到，自动传送门撤离（POE传送门）
```

### 3.2 倒计时机制详解

#### 示例1：基础塔体验
```
塔配置：
- 防御力：30
- 耐久：20分钟

玩家体验：
- 1-30层：无压力，慢慢打，收集材料
- 31层开始：倒计时20分钟开始，塔开始被怪物攻击
- 目标：在20分钟内尽可能爬高（目标40-50层）
```

#### 示例2：神话塔体验
```
塔配置：
- 防御力：90
- 耐久：70分钟

玩家体验：
- 1-90层：无压力，稳定推进
- 91层开始：倒计时70分钟开始
- 目标：冲刺100层神明，获得神器
```

### 3.3 自动撤离机制

```
倒计时结束时：
1. 自动出现传送门（POE经典传送门）
2. 玩家有5秒进入传送门
3. 5秒后塔倒塌，玩家强制传送回城
4. 保留所有收益（装备、材料、印记）
```

### 3.4 难度曲线设计

#### 螺旋上升机制
```
难度 = f(层数, 塔防御力, 塔材质)

- 层数越高 → 怪物越强
- 塔防御力越高 → 吸引的怪物越强
- 好的塔材质 → 需要更高层的材料 → 需要爬更高 → 难度更高

结果：螺旋上升的难度曲线
```

## 🎁 四、奖励系统

### 4.1 关键层奖励

```
每10层一个BOSS + 奖励：

第10层：材料包 + 普通装备
第20层：材料包 + 稀有装备
第30层：材料包 + 稀有装备 + 印记
第40层：材料包 + 传奇装备 + 印记
第50层：材料包 + 传奇装备 + 高级印记
...
第90层：大量材料 + 神话装备 + 稀有印记
第100层：神器 + 赛季专属装备 + 神级印记
```

### 4.2 装备制作系统（印记系统）

#### 印记概念（类似POE2第一赛季深渊物品）
```
塔内掉落特殊物品：
- "神罚印记"
- 可以影响装备词缀的生成权重
```

#### 使用方式
```
神罚印记 + 装备 + POE通货 → 定向词缀

示例：
- 火焰印记 + 戒指 + 混沌石 → 增加火焰词缀权重
- 暴击印记 + 武器 + 崇高石 → 增加暴击词缀权重
- 生命印记 + 护甲 + 改造石 → 增加生命词缀权重
```

#### 印记类型（示例）
```
元素系列：
- 火焰印记：影响火焰伤害词缀
- 冰霜印记：影响冰霜伤害词缀
- 闪电印记：影响闪电伤害词缀

战斗系列：
- 暴击印记：影响暴击相关词缀
- 速度印记：影响攻速/移速词缀
- 精准印记：影响命中/穿透词缀

防御系列：
- 生命印记：影响生命/回复词缀
- 护甲印记：影响护甲/防御词缀
- 抗性印记：影响元素抗性词缀
```

### 4.3 材料获取量级（示意）

```
官方塔（100层）一次完整攀登：
- 碎片：约2000-3000个（足够建2-3座基础塔）
- 精华：约150-200个（足够建1-2座强化塔）
- 核心：约15-20个（足够建1-2座神话塔）
- 印记：约10-20个（各类型随机）
```

## 🔄 五、游戏循环设计

### 5.1 螺旋上升循环

#### 第1圈：入门阶段
```
1. 爬官方塔（初始塔，无倒计时）
   → 获得材料（碎片为主）
   
2. 建造基础塔（防御30，耐久20分钟）
   
3. 攀登到40-50层
   → 获得装备和印记
   → 提升角色实力
```

#### 第2圈：进阶阶段
```
4. 继续爬官方塔
   → 获得更多材料（精华为主）
   
5. 建造强化塔（防御60，耐久40分钟）
   
6. 攀登到70-80层
   → 获得更好装备
   → 收集高级印记
   → 继续提升实力
```

#### 第3圈：终极挑战
```
7. 爬官方塔到顶
   → 获得核心材料
   
8. 建造神话塔（防御90，耐久70分钟）
   
9. 挑战100层神明
   → 获得神器
   → 获得赛季专属装备
```

### 5.2 策略选择

#### 保守流（稳定收益）
```
- 建造高防御塔（防御90）
- 慢慢打，稳定获取所有奖励
- 适合：做装备党，追求完美词缀
```

#### 激进流（快速刷刷刷）
```
- 建造低防御塔（防御30）
- 快速冲刺，只拿关键层奖励
- 适合：刷装备党，追求掉落数量
```

#### 平衡流（效率最优）
```
- 建造中等塔（防御60）
- 平衡速度和收益
- 适合：大多数玩家
```

### 5.3 动态补偿机制

```
装备好的玩家：
- 即使塔防御力低（基础塔）
- 也能快速冲到高层
- 获得关键层奖励
- 相当于"跳级"机制

装备差的玩家：
- 建造高防御塔（神话塔）
- 有更多时间慢慢打
- 稳定获取所有层奖励
- 逐步提升实力
```

## 🎯 七、设计意图总结

### 7.1 满足两类玩家

#### 刷装备党
```
玩法路径：
1. 快速建塔（基础塔）
2. 快速攀登（只打关键层）
3. 大量掉落
4. 重复多次

核心体验：刷刷刷的爽快感
```

#### 做装备党
```
玩法路径：
1. 收集印记（各类型）
2. 定向制作词缀
3. 结合POE通货
4. 精准打造装备

核心体验：可控性制造的成就感
```

### 7.2 长期留存机制

```
1. 材料收集：需要多次攀登官方塔
2. 塔的升级：从基础 → 强化 → 神话
3. 印记收集：收集各类印记，打造完美装备
4. 极限挑战：挑战100层神明
5. 赛季排行：最高层数排行榜
```

### 7.3 操作感与沉浸感

```
1. 向上攀登：视觉上的征服感
2. 倒计时压力：紧张刺激的体验
3. 自动撤离：魂类的"逃生"感觉
4. 神明战斗：史诗级的BOSS战
5. 塔的建造：策略选择的深度
```

### 7.4 复杂度控制

```
易于描述：
"建塔爬塔，在塔倒之前爬得越高越好"

低门槛：
- 官方塔无门槛
- 基础塔材料容易获得

有深度：
- 塔的选择（防御力 vs 耐久）
- 印记系统（定向制作）
- 策略选择（保守 vs 激进）
```

## 🎨 八、核心创新点

### 8.1 五大创新

1. ✅ **向上攀登**：区别于现有向下/横向系统，视觉征服感
2. ✅ **倒计时机制**：增加紧张感和策略深度，类魂逃离体验
3. ✅ **建造系统**：玩家主动选择难度和策略，深度玩法
4. ✅ **印记制作**：结合POE通货，定向打造装备，满足做装党
5. ✅ **螺旋难度**：自我迭代的难度增强，长期挑战目标

### 8.2 符合POE风格

```
✅ 材料系统：碎片/精华/核心（类似POE通货体系）
✅ 兑换建造：NPC兑换（类似POE制作台）
✅ 印记系统：影响词缀（类似POE2深渊物品）
✅ 自动传送：传送门撤离（POE经典机制）
✅ 简洁清晰：只有2个核心属性（防御力+耐久）
```

## 📊 十、设计会议记录

### 10.1 团队配置
- **策划主导**：负责整体方向和玩法设计
- **系统设计师**：负责建造系统和循环设计
- **数值设计师**：负责难度曲线和奖励平衡

### 10.2 关键决策

#### 决策1：采用向上攀登
```
原因：
- 区别于现有深渊（向下）和试炼（横向）
- 视觉上更有征服感
- 符合"猎神"的主题
```

#### 决策2：引入倒计时机制
```
原因：
- 增加紧张感和挑战性
- 提供类魂的逃离体验
- 创造策略深度（速度 vs 稳定）
```

#### 决策3：简化塔的属性为2个
```
原因：
- 符合POE简洁风格
- 易于理解和传播
- 降低新手门槛
```

#### 决策4：印记系统结合POE通货
```
原因：
- 满足做装党的可控性需求
- 与现有系统融合
- 增加装备制作深度
```
