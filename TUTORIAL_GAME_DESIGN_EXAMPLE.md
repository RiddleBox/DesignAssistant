# 数字员工协作系统 - 使用教程

## 📚 教程概述

本教程将以**设计一个RPG游戏的战斗系统**为例，完整演示如何使用数字员工协作系统，从零开始搭建团队、分配任务、协作完成项目。

**教程目标**：
- 学会使用7个核心SKILL
- 理解完整的协作工作流
- 掌握系统的核心功能

**预计时间**：30分钟

---

## 🎮 案例背景

**项目名称**：RPG游戏战斗系统设计  
**项目目标**：设计一个回合制战斗系统，包括战斗流程、技能系统、数值平衡  
**团队配置**：系统策划、数值策划、战斗策划
